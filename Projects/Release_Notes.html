Purpose

The FP-ATR-SIGFOX1 Projects provide examples and applications to aid understanding how to develop Asset Tracker applications for NUCLEO-L072RZ and NUCLEO-L476RG. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
